# Task Manager Backend

This is the backend repository for a task management application developed as a solo project using CodeIgniter.

The API powers a task management system inspired by modern project management platforms such as ClickUp. It provides endpoints for creating, retrieving, updating, and deleting tasks while enforcing business rules and managing data persistence.

The project is designed to showcase backend development practices, including RESTful API design, database interaction, validation, and application architecture using CodeIgniter.

## Tech Stack

* PHP
* CodeIgniter
* MySQL
* REST API
